## **インストール**:
インストールする際は、zip ファイルをダウンロード/展開後 "OreAddin" フォルダを **アドイン** として登録してください。

アドインの登録は、[こちら](https://kantoku.hatenablog.com/entry/2021/02/15/161734)の手順に従ってください。
